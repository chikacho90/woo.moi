# woo.moi

Creative developer portfolio — Next.js + React + Tailwind CSS + Three.js

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- `/` — public landing page (shader art)
- `/do` — project & task management (hidden, requires unlock)
- `/think` — insights & knowledge feed (hidden, requires unlock)
- `/gallery` — visual works (hidden, requires unlock)
- `/ootd` — weather-based outfit recommendations (hidden, requires unlock)
- `/projects` — project dashboard (hidden, requires unlock)
- `/labs` — R&D experiments (hidden, requires unlock)

Hidden sections are revealed by unlocking the secret gate on the main page.
